# Expense Tracker (Console-Based)

A simple console-based Expense Tracker built using Python.  
This application allows users to manage daily expenses with features like adding, viewing, deleting, filtering, and calculating total spending.

---

## Features

- Add new expenses
- View all expenses
- Delete expenses by ID
- Filter expenses by category
- Calculate total spending
- Data stored using JSON file (persistent storage)

---

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- JSON File Handling
- List Comprehension
- Exception Handling
